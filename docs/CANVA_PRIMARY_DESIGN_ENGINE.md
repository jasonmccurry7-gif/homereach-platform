# HomeReach Canva Primary Design Engine

## Objective

Canva is the primary visual execution layer for HomeReach. HomeReach remains the source of truth for campaigns, route logic, pricing, approvals, payments, and production state. Codex/HomeReach generates strategy, structured content, template fields, safety checks, and export requests.

## Integration Path

The recommended production path is:

1. Create a Canva Developer integration using Canva Connect APIs.
2. Configure OAuth using `CANVA_CLIENT_ID`, `CANVA_CLIENT_SECRET`, and `CANVA_REDIRECT_URI`.
3. Apply migration `092_canva_primary_design_engine.sql`.
4. Set `CANVA_TOKEN_ENCRYPTION_KEY` so OAuth tokens can be stored encrypted server-side.
5. Create HomeReach Brand Templates in Canva and add their IDs to Vercel env vars.
6. Use HomeReach admin page `/admin/canva` to inspect readiness and template mapping.
7. Use `/api/admin/canva/autofill` to turn structured campaign/postcard/proposal data into Canva designs.
8. Use `/api/admin/canva/exports` to create export jobs for approved Canva designs.

## Environment Variables

Required for OAuth:

- `CANVA_CLIENT_ID`
- `CANVA_CLIENT_SECRET`

## Template Strategy

HomeReach should keep Canva templates editable and data-driven:

- Presentations use Brand Templates and Autofill.
- Postcards use front/back Brand Templates with fields for headline, body, CTA, QR URL, disclaimer, image direction, and campaign/customer metadata.
- Map visuals should be generated by HomeReach maps and uploaded or embedded as image fields.
- Print-ready exports require human approval and quote lock.

## Safety Boundary

No Canva export should become a production print file until:

- source data is verified
- USPS route counts are verified
- print and postage math is locked
- legal/disclaimer review is complete
- campaign/client approval is complete
- HomeReach admin approval is complete

## Official Canva Capabilities Referenced

- Canva Connect APIs
- OAuth authentication
- Brand Templates
- Autofill jobs
- Export jobs
- Asset and folder management

Note: Canva Brand Templates and Autofill are Canva Enterprise-only capabilities per Canva's Connect API docs. HomeReach should keep dry-run planning enabled until the connected Canva workspace supports those endpoints and template IDs are configured.
