## Review Checklist

- [ ] Competitor name/address is clear.
- [ ] Duplicate locations removed.
- [ ] Radius is reasonable for the ad budget.
- [ ] Broader fallback geography exists if the platform blocks targeting.
- [ ] Campaign copy avoids surveillance language.
- [ ] No individual-level private identity is exposed.
- [ ] No sensitive-trait targeting is used.
- [ ] No guaranteed lead, sales, visit, ROI, or conversion claim is included.
- [ ] Client approval is recorded.
- [ ] Admin approval is recorded before launch.

## Approved Language

```text
We can help you stay visible in the local areas where homeowners are already comparing options.
```

```text
This campaign focuses on geography-based visibility around competitor-heavy service areas where platform rules allow.
```

## Blocked Language

Do not use:

- Spy on competitors.
- Track their customers.
- Follow people from competitor locations.
- Steal customers.
- Guaranteed competitor conquest.

## Recommended Fallbacks

If a competitor location cannot be used directly, use one of:

- Nearby ZIP code.
- City or county.
- Service-area radius.
- Neighborhood or route cluster.
- Direct mail route review.
