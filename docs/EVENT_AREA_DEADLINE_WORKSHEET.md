## Review Checklist

- [ ] Event name and location are clear.
- [ ] Event date is confirmed.
- [ ] Event source/details are confirmed by the client.
- [ ] Launch cutoff is feasible.
- [ ] Rush review fee is approved if needed.
- [ ] Target geography/radius is reasonable.
- [ ] Campaign offer is approved.
- [ ] Landing/tracking destination is approved.
- [ ] Creative is approved.
- [ ] No guaranteed attendance, sales, leads, ROI, or platform approval claims are included.
- [ ] No event-attendee tracking or individual-level surveillance language is included.
- [ ] Client approval is recorded.
- [ ] Admin approval is recorded before launch.

## Approved Language

```text
We can help your business stay visible around the event area during the promotion window.
```

```text
This campaign is designed for time-sensitive local visibility. Results vary and platform approval is required.
```

## Blocked Language

Do not use:

- Guaranteed attendance.
- Guaranteed sales.
- Track event attendees.
- Follow attendees after the event.
- Guaranteed event ROI.
